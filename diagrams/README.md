# Integration Architecture Diagram

Folder ini berisi diagram arsitektur integrasi sebagai asset terpisah dari dokumen laporan.

| File | Fungsi |
|---|---|
| `integration-architecture.png` | Gambar utama untuk dikirim atau dimasukkan ke laporan |
| `integration-architecture.svg` | Versi SVG |
| `integration-architecture.drawio` | Versi editable untuk diagrams.net / draw.io |

## Cara Edit di diagrams.net

1. Buka `https://app.diagrams.net`.
2. Pilih `Open Existing Diagram`.
3. Pilih file `integration-architecture.drawio`.
4. Edit diagram sesuai kebutuhan.
5. Export ke PNG/SVG jika perlu.

## Referensi Bentuk Diagram

Diagram dibuat mengikuti gaya Enterprise Application Integration yang menampilkan:

- aplikasi enterprise dan database masing-masing,
- message channel / broker,
- subscriber queue,
- adapter / endpoint,
- router,
- translator,
- outbound adapter,
- format data dan protokol integrasi.
